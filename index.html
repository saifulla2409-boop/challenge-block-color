How to play

  1. Hit start to spawn a fresh grid of colorful blocks.
  2. Match the block that exactly matches the preview color.
  3. Correct picks add time and score. Wrong picks cost a life.
  4. Level up every three correct matches — the grid grows!

Pro tip: tap Hint to briefly highlight the correct block. Use sparingly — it costs points.
